if else statement :- An if...else statement in JavaScript lets you run different blocks 
of code depending on whether a condition is true or false.
syntax
if(expression)
{
    statement
}
else
{
    statement
}

switch statement 
switch statement is used when you want to run diffrent block of code based on 
single expression
example:  switch (expression) {
  case value1:
    // Code if expression === value1
    break;

  case value2:
    // Code if expression === value2
    break;

  default:
    // Code if no case matches
}
