what is array 
In JavaScript, arrays can store elements of different data types within the same array. This is a key characteristic 
of JavaScript's dynamic typing, meaning you are not restricted to storing only one type of data in an array, unlike
 in some other programming languages. For example, a single JavaScript array can contain numbers,
 strings, booleans, objects, or even other arrays: JavaScript

all array function 
********Mutator Method (Change the original array)**********
push()=> add element at the end(Stack)
pop()=> remove element from end(Stack)
shift()=> remove first element (queue)
unshift()=> add element first(queue)
splice()=> add/remove frm specific index
sort()=> sort the array
reverse()=> reverse the array
copyWithin()=>copy part of array to another location in the same array

Accessor Method
concat() => merge array
slice() => return a portion of array 
include() => check if array include this value
indexOf()  => find first index of value
lastIndexOf()  => find last index
toString() => convert array to string
at() =>get element at specific index
